  • defaultTemporaryDirectory

    The name of the directory used for temporary files. This defines the default value for the directory argument to all functions involving a temporary path.

    Pathos calls searchForDefaultTemporaryDirectory to calculate this value. The result of this search is cached. You can use the same method to reset this value if needed.

    Declaration

    Swift

  • searchForDefaultTemporaryDirectory()

    Searches a standard list of directories to find one which the calling user can create files in. The list is:

    • The directory named by the TMPDIR environment variable.
    • The directory named by the TEMP environment variable.
    • The directory named by the TMP environment variable.
    • The directories /tmp, /var/tmp, and /usr/tmp, in that order.

    • As a last resort, the current working directory.

    Declaration

    Swift

    public func searchForDefaultTemporaryDirectory() -> String

    result of the search.
