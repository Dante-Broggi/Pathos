  • defaultTemporaryDirectory

    The name of the directory used for temporary files. This defines the default value for the directory argument to all functions involving a temporary path.

    Pathos calls searchForDefaultTemporaryDirectory to calculate this value. The result of this search is cached. You can use the same method to reset this value if needed.
